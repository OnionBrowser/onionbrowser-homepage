Onion Browser has successfully connected over Tor.

This homepage is a secure "onion site" which is only accessible over Tor. You can further check your Tor connection status here.

Onion Browser 2.3.0 was released on December 17, 2019. It fixes an OCSP-related leak, features a redesigned settings page, and contains many more changes + fixes. Please update in the App Store.

For the latest information about Onion Browser, see the Onion Browser site.

Due to iOS limitations, WebRTC and media files leak outside of Tor and are routed over the normal internet. This will reveal your real IP address (or your VPN's address) to sites using these features. To defend against this, disable Javascript by setting the security level to "Strict". More information about these types of leaks can be found here.
